# DEC-0007: Use isolated worktrees and sandboxed execution for the owned runtime

## Decision
Start the 0.7 supervised runtime with one writer per task, a dedicated Git worktree, bounded budgets, and Docker-recommended execution for untrusted repositories.

## Rationale
- Repository code and agent actions require containment.
- A worktree makes patch review, rollback, and trace boundaries explicit.
