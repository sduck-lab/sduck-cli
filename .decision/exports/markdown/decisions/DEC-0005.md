# DEC-0005: Make 0.6 a reliable MCP control plane before owning execution

## Decision
Deliver typed, idempotent MCP operations and CI validation before adding a sduck-owned model loop or sandbox runtime.

## Rationale
- MCP makes the existing lifecycle conversational with lower risk.
- Approval, privacy, and mutation consistency must be correct before agent execution is introduced.
