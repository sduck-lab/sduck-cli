# DEC-0031: Adopt canonicalize for RFC 8785 digest serialization

## Decision
Use the canonicalize package default ESM export for RFC 8785 JCS serialization; hash its UTF-8 output with node crypto SHA-256 and reject non-JSON-canonicalizable values.

## Rationale
- RFC 8785 Appendix G lists canonicalize.
- It is TypeScript/ESM-ready with no runtime dependencies and avoids maintaining a hand-rolled canonicalizer.
