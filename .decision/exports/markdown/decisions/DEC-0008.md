# DEC-0008: Treat hooks as advisory and enforce workflow in CI

## Decision
Use optional editor hooks for ergonomics only; validate approved briefs and trace coverage with a CI-facing sduck verify command.

## Rationale
- Hooks can be bypassed by shell, IDE, alternate tools, or control-plane edits.
- CI and branch protection are the enforceable repository boundary.
