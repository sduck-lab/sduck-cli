# DEC-0049: Treat LLM handoff and Git evidence differently

## Decision
The skill classifies decisions directly stated in the LLM handoff as EXPLICIT only when corroborated by the user or durable source; patch-only conclusions stay INFERRED with conservative confidence. It asks follow-up questions for unsupported rationale.

## Rationale
- Git diff establishes what changed, not why.
- LLM conversation can be sensitive and may contain unverified assumptions.
