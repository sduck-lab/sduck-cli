> **이 작업은 외부 동작(기능, API 응답, UI)을 변경하지 않는다.**

## 4. 리팩토링 범위

### 변경 대상 파일

```
target_paths:
  - src/...
  - tests/...
```

### 변경하지 않는 범위 (명시적 경계)

```
blocked_paths:
  - src/api/routes/    # 외부 API 계약 불변
  - migrations/        # DB 스키마 불변
```

### 인터페이스 불변 확인
